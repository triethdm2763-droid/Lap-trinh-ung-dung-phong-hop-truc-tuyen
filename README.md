UDM_03 Lập trình ứng dụng Phòng họp trực tuyến

HƯỚNG DẪN SỬ DỤNG HỆ THỐNG PHÒNG HỌP TRỰC TUYẾN

1. Tham gia phòng (Join Room)
Nhập Room ID do Host cung cấp
Nhập Username để tham gia phòng
Sau khi vào phòng, bạn có thể bắt đầu chat

2. Chat công khai (Public Chat)
Gõ nội dung và nhấn Enter
Tin nhắn sẽ gửi đến tất cả mọi người

Ví dụ: Hello mọi người 😄

3. Chat riêng (Private Chat)
Cú pháp: private <username> <message>

Ví dụ: private An Hello bạn 🤫

->Chỉ bạn và người nhận thấy tin nhắn

4. Gửi file
Cú pháp:
file <đường_dẫn_file>

Ví dụ: file C:\Users\Admin\Desktop\test.txt

->File sẽ được gửi trong phòng và sẽ download sẵn về máy

5. Sử dụng Emoji

Bạn có thể gõ emoji trực tiếp:
Hello 😄
Good job 👍
Ok nhé 🔥

✨ Một số emoji phổ biến
    Ký hiệu	   Emoji	  Ý nghĩa
    :)	        😊       Vui vẻ
    :D	        😄	     Cười tươi
    :>	        😂	     Cười lớn
    :(	        😢	     Buồn
    <3	        ❤️	     Yêu thích
    :fire	      🔥	     Cháy
    :thumbsup	  👍	     Đồng ý
    :cool	      😎	     Ngầu

6. Các lệnh cơ bản
Lệnh	Chức năng
exit	Thoát chat
private	Chat riêng
file	Gửi file

7. Chức năng dành cho Host
Kick user 
Đóng phòng 
Quản lý người tham gia 

8. Lưu ý
Gõ đúng username khi chat riêng
Phải cùng phòng mới private được
File phải tồn tại
